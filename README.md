# Binance 批量提币工具

> 📢 **关注 Twitter: [@Nadiinn5](https://twitter.com/Nadiinn5)** 获取更多机器人和工具！

一键批量提币，支持多地址、多币种，全自动化操作。

## 🚀 快速开始

### 方式一：直接运行（需要 Node.js）

```bash
npm install
node server.js
```

### 方式二：使用打包好的 EXE（推荐）

直接双击 `打包/` 目录下的 `提币工具.exe` 即可运行。

### 访问工具

打开浏览器访问: **http://localhost:7754**

## 📋 功能特点

- ✅ 批量提币：一次提多个地址
- ✅ 多币种支持：USDT、USDC、ETH 等
- ✅ 多网络支持：TRC20、ERC20、BEP20 等
- ✅ 本地运行：API Key 不上传服务器
- ✅ 服务器时间同步：避免时间戳错误

## ⚠️ 安全提醒

1. **API Key 设置**：
   - 必须开启提现权限
   - 建议设置 IP 白名单
   - 必须设置地址白名单

2. **使用注意**：
   - 请勿在公共网络运行
   - 使用完毕后关闭服务
   - 定期检查 API Key 权限
