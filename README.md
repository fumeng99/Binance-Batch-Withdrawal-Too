# 🏦 Binance 批量提币工具 v3.1

## ⚡ 功能

- 📦 **批量提币** - 多地址一键处理，告别逐个操作
- 💰 **多币种支持** - USDT、USDC、ETH、BNB、BTC 等主流币种
- 🌐 **动态网络** - 自动获取币种支持的所有网络
- 💵 **费用预估** - 实时显示手续费及 USD 估值
- ⏳ **智能延迟** - 固定或随机延迟，更自然防监控
- 🔄 **失败重试** - 失败任务单独列出，一键重试
- 🌍 **代理支持** - 网页内设置代理，支持检测 IP 和国家
- 🔔 **音效提醒** - 成功/失败音效提示

## 📖 使用方法

1. 🚀 运行 `node server.js` 启动服务
2. 🌐 浏览器访问控制台显示的地址
3. 🔐 粘贴控制台的 **Session Token**
4. 🔑 填写 **API Key** 和 **Secret Key**
5. 💱 选择**币种**和**网络**
6. 📝 输入提币列表
   - 格式：`地址,数量`（每行一个）
   - 或使用「统一数量」模式，每行只输入地址
7. ▶️ 点击「🚀 开始批量提币」

## 🌍 代理配置

在网页「API 配置」中填写代理 URL，支持以下格式：

```
host:port
host:port:user:pass
user:pass@host:port
http://host:port
http://user:pass@host:port
socks5://host:port
socks5://user:pass@host:port
```

点击「🔍 检测」可测试代理连通性并显示 IP 和国家。

## ⚠️ 注意事项

- ✅ API Key 需开启**提现权限**
- ✅ 提币地址需加入 Binance **白名单**
- ✅ 建议设置 **IP 白名单**
- 🚫 请勿在公共网络运行
- 🔒 使用完毕后关闭服务

## 🔒 安全特点

- 🛡️ **三层加密保护** - HTTPS + 传输层AES + 应用层AES
- 🔑 **随机密钥** - 每次启动自动生成新密钥
- 💻 **本地运行** - 仅监听 127.0.0.1，API Key 不上传
- ⏱️ **时序安全** - timingSafeEqual 防止时序攻击
- 📋 **请求追踪** - 每个请求生成唯一 ID
